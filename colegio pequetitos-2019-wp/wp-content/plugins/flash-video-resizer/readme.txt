Resize flash and HTML5 video to a specific size without having to edit HTML code.

== Description ==

English:
This plugin resizes flash and HTML5 videos using a width specified in WordPress's Admin Panel. There is an different configuration for mobile sites, but requires [WordPress Mobile Pack] (http://wordpress.org/extend/plugins/wordpress-mobile-pack/ "WordPress Mobile Pack")
.
Tested flash video services listed below.

Português:
Este plugin redimensiona videos em flash e HTML5 para uma largura determinada por uma opção no painel de administração do WordPress. Existe uma configuração separada para sites para celulares, mas requer o plugin [WordPress Mobile Pack] (http://wordpress.org/extend/plugins/wordpress-mobile-pack/ "WordPress Mobile Pack")
.
Testado com os serviços listados abaixo

== Installation ==

This section describes how to install the plugin and get it working:

1. Upload plugin's folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Adjust video width in the Reading section of WordPress configuration

Como instalar o plugin:

1. Faça o upload da pasta do plugin para o diretório `/wp-content/plugins/` do seu blog.
2. Ative o plugin através do menu 'Plugins' no seu WordPress.
3. Ajuste a largura do vídeo na seção Leitura da página de administração do seu WordPress.
